## Master Finder & File System Navigation

### Finder Tips & Shortcuts

1. **Cmd + N** — Open a new Finder window.
2. **Cmd + Shift + N** — Create a new folder.
3. **Cmd + Delete** — Move selected file or folder to Trash.
4. **Cmd + Shift + Delete** — Empty Trash.
5. **Cmd + ↑** — Go to the parent folder.
6. **Cmd + ↓** — Open the selected folder or file.
7. **Cmd + Option + P** — Show or hide the path bar.
8. **Cmd + /** — Show or hide the status bar.
9. **Cmd + Shift + .** — Show or hide hidden files (e.g. `.git`, `.env`).
10. **Spacebar** — Quick Look preview for files.
11. **Cmd + F** — Search within Finder.
12. **Cmd + Option + S** — Show search criteria (filters).
13. **Cmd + 1 / 2 / 3 / 4** — Switch Finder views (Icon, List, Column, Gallery).
14. **Right-click → Get Info** — View file size, permissions, and metadata.
15. **Tags** — Use color tags to organise files by priority or type.
16. **Sidebar favourites** — Pin frequently used folders for quick access.
17. **Column View (Cmd + 3)** — Quickly navigate deeply nested folders.
18. **Sort by Kind / Date / Size** — Helps locate files faster.
